## compile and build with maven in terminal  ##
~~~
cd //directory of the project
~~~
~~~
mvn clean install
~~~

## Execute the game in terminal ##

~~~
cd //directory of the project
~~~
~~~
java -cp ./Target/TheMazingGame-0.0.1-SNAPSHOT.jar main.java.fr.ul.acl.Mazing.TheMazingGame.start.Main
~~~
## Tutoriel  ##
~~~
Right click -----> "RIGHT"
Left click -----> "LEFT"
"UP" click -----> "JUMP"
"Z" click ----> "Simple attack"
"E" click ----> "Special attack"
"R" click ----> "far-distance attack"
"F" click (in gate) -----> "NEXT LEVEL"
~~~
##  Objectif ##
~~~
Kill all monsters to spawn the key and finish all the levels to win the Game.

~~~
